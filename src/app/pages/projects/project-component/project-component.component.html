PLANT CARE

Plant care foi um projeto desenvolvido para a matéria de pesquisa e inovação para a faculdade Bandtec.

Github do projeto

Basicamente, consiste em um vaso de planta que pode ser utilizado para monitorar pessoas de idade. Através de sensores de temperatura e umidade nós conseguimos dar emoções a planta, motivando o idoso a cuidar da mesma. Todos os dados de interação com a planta são apresentados em um dashboard especial para o cuidador.
